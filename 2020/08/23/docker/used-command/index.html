2020年08月23日 168 字 大概 1 分钟

docker常用命令

描述 命令
删除所有停止的容器 docker rm $(docker ps -a -q)
强制删除所有镜像 docker rmi -f $(docker images -q)
设置容器自启 docker container update –restart=always 容器名称
进入容器 docker exec -it 容器名称或ID bash
进入容器 docker exec -it -u root 容器名称或ID /bin/bash
删除所有名称为none的镜像 docker images|grep none|awk ‘{print $3}’|xargs docker rmi
拷贝文件至docker容器 docker cp /home/mysql/conf/my.conf  容器名称或ID:/etc/mysql
构建镜像 docker build -t mysql8-xtrabackup:1.0.0 .
上传镜像 docker push mysql8-xtrabackup:1.0.0
修改镜像名称 docker tag 镜像ID 新名称:新版本
